# 8oot Logo Changer

TIP

Not maintained anymore

# Description

With this tool you can change the Windows 8 / Windows 8.1 Boot Logo picture.

Author: Victor Alberto Gil

# Features

# What is it?

8oot Logo Changer is a tool to that will help you to change Windows 8 / Windows 8.1 boot logo with a custom picture, it has the following features:

# Simple Editor

You can open or drop any picture in the application and select the area to use as the boot logo.

# How to use boot logo editor

Just open any supported picture in the editor, using either the open file dialog or just dropping a picture file, select the desired area on the picture and create the bitmap files with the button [>]

After that, generate the bootres.dll file, it will be located in a temporary directory (inside %APPDATA%).

Once ready, Apply it, it will ask you to install a local certificate that it needs to be present to sign bootres.dll created, otherwise it won't work, because it needs a "valid" certificate to be loaded at boot time. When a dialog asking for Smart Card shows up, just cancel it, and then accept to install that generated local certificate. This will be asked only one time since future modifications will reuse this certificate.

# Watermark remover

As this tool requires you to set TESTSIGNING mode ON in order to be able to show the new boot logo on boot, Windows will show a watermark showing the Test Mode status, this is normal, but nobody wants it, so this tool also includes a patcher that will modify two MUI file belonging to its correspondent DLL files: shell32.dll.mui and basebrd.dll.mui - Don't worry, this won't alter shell32.dll nor basebrd.dll.

# What does it exactly do?

# Patching Bootlogo

It replaces 6 BMP files inside bootres.dll, those are pictures with fixed sizes using a 24bit color depth. This tool adjusts the selected picture according to original sizes inside that dll, and it changes from 32bit to 24bit if necessary. After that, it replaces those pictures insider bootres.dll.

When it applies those changes, it is recommended to make a backup (which will be located inside the same directory i.e. C:\Windows), that backup is the original bootres.dll, which later we can use to restore it.

# Patching the MUI files

There are two files that we neew to modify in order to hide the watermark shown by Windows if we are on TEST MODE.

  • shell32.dll.mui is located inside your system32 directory, specifically inside a localization folder (e.g. en-US), this file has resource strings table, which is used to show that message in the desktop according to your system language, so this tool will make those changes for you.

  • basebrd.dll.mui is another file which is located inside Windows\Branding\Basebrd directory, then again inside the localization folder. And it also holds part of the message that Windows shows as a watermark when we are in Testsigning mode (TEST MODE).

As you can see in the picture, it will show those message in the tool (visible if not patched).

# Finally

Thos are the changes made using this tool, but if you want to learno more about how it can be done manually or using other tools, please refer to this link -> Winmatrix by Krutonium

# Pros

  • Crops and Auto adjusts the bitmap size.
  • A simple simulated previewer that might help you decide which picture to use.
  • Reversible, any changes can be restored.

# Cons

  • Tested only on Windows 8/8.1 Pro x64.
  • It might present incompatibilities with other MUI patches that you might have done using other tools.
  • UEFI not supported.
  • It might be necessary to restore backup before Windows Update, in case this update needs to replace those files. Recommended to restore before that.

# Requirements

  • Windows 8 / Windows 8.1 x32 or x64
  • 5 MB RAM minium
  • Disk Space: 8MB
