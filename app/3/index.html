# WinXCorners

Platform

Windows 10 x86/x64

Version v1.1

Type Closed Source

# Description

WinXCorners allows you to assign events to trigger on your main monitor's screen corners, it activates when your mouse cursor hovers them, i.e. when you move your mouse cursor to your screen edges.

# Features

  • Portable
  • No hooks
  • Windows 10 GUI like appearance
  • Accent Color support
  • Corners with different options
    • Show all windows (just like pression Win+Tab)
    • Show Desktop
    • Start Screen Saver
    • Turn Off Monitor
    • Show Action Center
    • Hide other windows
  • Set delay time for each corner before executing action
  • Optionally shows an animated countdown
  • It can also be added a custom command (with parameters) as an action
  • Launch on startup option
  • No need to save settings, it will autosave any change.

# How to use

# Installation ?

You don't need to install it, since it is portable.

Extract files in any writeable directory, launch it and you will find the application in the system's tray area.

Left click to launch it, right click for contextual menu.

It is recommended to enable Start with Windows option in the contextual menu, it will launch this application as soon you're logged in Windows 10.

To "uninstall", just disable Start with Windows option.

NOTICE Windows 10 normally hides icons from the systray, application icon still is located in the Show hidden icons, accessible by clicking the button with upward arrow icon in the system tray. You can drag WinXCorner icon to the systray if you intend to access it faster. Recommended if you are going to disable it, maybe when playing games or using a full screen app like watching movies.

# Usage

Left click its icon on the systray. You will see, for dropdown options, they represent each screen's corner, just select any action you like, and there is also an empty option which means, no action will be done on that hot corner.

It is also a toggle option to disable temporarily, useful if you don't want to activate actions on corners when playing games for instance.

# Advanced options

Right the application's icon, and choose Advanced.

You can set a global delay timer before hot corners executes the assigned action. You can also specify each corner's delay time.

Timer uses unit in 300ms interval, i.e. if you set to 3 it will be 900 milliseconds, almost a second waiting time before action is executed.

A countdown animation is also supported, by checking that option, you will see an animation on the hot corner, it gives a visual hint before executing the action assigned. Useful if you moved your mouse accidentally and it gives time to move it out in order to avoid executing the assigned action.

# Custom Command

Also known as launcher, if you enable the Custom Command option (check), the command specified in the first text edit field will be executed, and it has optionally a parameters field.

To assign a hot corner to this custom command, proceed as normal. Left click the application's icon, there in the corner's dropdown option, you will now see Custom Command option, and select it.

# Pros

  • Native application, faster load on cold boot compared to .Net applications.
  • Simple and easy to use.

# Cons

  • It won't work while you are using (focused window) an elevated application. But it will try, and sometimes will work. A workaround would be to click the taskbar before moving mouse's cursor to any hotspot.
