# Windows 8 Start Screen Customizer

TIP

Not maintained anymore

# Description

With this tool you can change the Windows 8 Start Screen background picture with your own picture.

Author: Victor Alberto Gil

# Features

# What is it?

Windows 8 Start Screen Customizer is a simple software that will allow you to customize the Start Screen background picture on Windows 8/8.1, it includes the following features:

# Gif Animations support

You can set any animated gif as your start screen bakcground, including bigger ones.

# Multimonitor

Auto adjusts to the monitor where the Start Screen is showing.

# Opacity

You can change the opacity of the Start Screen.

# Pros & Cons

  • Changing background picture immediately, no need to restart.
  • No changes are made to any important system files.
  • These changes are only running on memory, and will work as long as this software is running.
  • There is an option to run this application at startup so it will run on logon and apply as soon as it starts.

# Multilanguage

I've added some translations, not very accurate though. Bear in mind that I just used an online translator. Feel free to contact me and add a proper language translation.

# Requirements

  • Windows 8 / Windows 8.1 x32 or x64
  • 20 MB RAM minium (it will increase depending of the gif size)
  • Disk Space: 2MB
